Chips of Fury: the 'We Gotta Eat Too' update

Remember when they said 'nothing in life is free'? Well, our server bills agree. Starting with this update, we're introducing our PRO plan.

Here's how it works: you can try all premium features for free during the first 30 minutes of each game. After that, you can continue playing the basic game for free, or upgrade to PRO to keep access to all features.

Real talk? We're a bit nervous about this change. But between keeping the lights on and keeping the game running smoothly, we had to make this move sooner or later.

Your support means the world to us (and our landlords). This update is stacked with features, so if you spot any bugs trying to bluff their way in, give us a shout. We'll be fully focused on ensuring a smooth experience for everyone in the coming days.

Note: Paid plans are not available on the web and android APK yet, please download the app from play store or app store to purchase. After that you can continue using the app on any platform.

This release brings a major change in how chips are distributed. This is an attempt to simplify the flow from joining a game to buy-in

    Till the previous version
  • A player joined the game room using a code
  • Then to obtain chips they had to buy chips
  • The admin had to approve this request
  • The chips were credited to the player's wallet
  • Finally the player could proceed to buy-in

ğŸ‘® Now, only Admin can give (and take back!) chips

For a non-admin player joining the game, the flow will look like this

  • A player joins the game room using a code
  • The admin will credit chips to the player's wallet
  • The player can now buy-in

This means the admin has to do a bit more work. But it can be reduced by using the auto-credit chips setting in the game config

ğŸ’° Improved auto-credit behaviour

At the time of approving a join request, the admin will now also get an option to disable the auto-credit for that particular player
